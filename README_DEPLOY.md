# Déploiement (GitHub Pages)

Ce paquet contient un site statique (FR) pour la Fondation Intelligence : programme éducatif public « Littératie en sécurité de l’IA ».

## Contenu

- `index.html` : page principale (modules 0–5 publiés, journal, gouvernance)
- `docs/` : documents de gouvernance et extraits officiels (PDF) + kit CA Zoom
- `assets/` : images

## Déploiement

1) Copiez `index.html`, `docs/` et `assets/` à la racine du dépôt GitHub Pages.
2) Commit + push.
3) Vérifiez les liens (navigation, documents PDF/ZIP).
